# 🔬 超声波舌位视频数据库 — Ultrasound Tongue Imaging DB

用于语音学研究的超声波舌位影像存储、检索和播放系统。

## 功能

- 📹 上传超声波舌位视频（文件或链接）
- 🔤 按音素 (IPA)、语言、发音人分类
- 🔍 多条件筛选检索
- ▶️ 在线播放视频
- 💾 数据存储在浏览器 IndexedDB（本地）
- 📱 响应式设计

## Demo

[https://jingsun-tr.github.io/ultrasound-tongue-db/](https://jingsun-tr.github.io/ultrasound-tongue-db/)

## 使用

1. 填写视频信息（标题、音素、语言、发音人）
2. 上传视频文件或粘贴链接
3. 在上方筛选区按条件检索
4. 点击视频卡片播放

## 研究用途

- 超声舌位图像对比分析
- 发音教学材料库
- 多语言发音对比研究
